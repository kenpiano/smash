# SMASH

**SMASH** is text editor that assist software developers.

## CONSTITUTIONS

The basic value of SMASH project is just following 3 points.

1. Work fast.

2. Work everywhere.

3. Useful.

## BASIC REQUIREMENTS
The basic requirements of SMASH are as follows.
- Fast startup time (within 0.2 seconds on a typical machine).
- Low memory usage (less than 500KB on startup).
- Cross-platform compatibility (Windows, macOS, Linux).
- Support for LSP and syntax highlights.
- Support for large files (e.g., 100MB+).

## ADVANCED REQUIREMENTS
The advanced requirements of SMASH are as follows.
- Real-time collaboration features.
- Integrated terminal emulator.
- Advanced code navigation (e.g., go to definition, find references).
- Built-in debugging tools.
- Support for remote development (e.g., SSH, WSL).
- Customizable UI themes and layouts.
- Extensible through plugins.(optional)
- User-friendly interface with customizable keybindings.(optional)
